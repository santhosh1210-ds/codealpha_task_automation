# 🗂️ Task Automation with Python Scripts

## 📌 Project Overview
A Python automation script built as part of the CodeAlpha Python Programming Internship (Task 3). This script automates the repetitive task of moving all `.jpg` files from a source folder to a destination folder.

## 🛠️ Tools & Libraries
- Python 3
- `os` module
- `shutil` module

## 🔍 How It Works
1. User enters the source folder path
2. User enters the destination folder path
3. Script creates the destination folder if it doesn't already exist
4. Script scans the source folder for `.jpg` files
5. Each `.jpg` file found is moved to the destination folder
6. A summary of total files moved is displayed
